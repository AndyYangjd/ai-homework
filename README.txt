# ai-homework
machine learning homework
机器学习课程大作业：
实验目标:
给定一个可二分类的二维数据集，通过使用机器学习方法求出一个模型（即分类直线），具有良好的泛化能力，对测试集内的数据也进行正确分类。
项目结构：
AI-homework                           项目文件夹
-------+ main.py                      总流程控制
-------+ data.py                      准备数据集 
-------+ data_visual.py              数据可视化
-------+ perceptron_algor.py         感知机算法 
-------+ svm_algor.py                支持向量机算法
